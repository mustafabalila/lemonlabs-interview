## Lemonlabs interview

A simple API that interacts with [Twitter](https://twitter.com/).

##### You'll need to prepare a .env file check out the .env.example

## Install packages

```zsh
$ yarn
```

## Run the build script

```zsh
$ yarn build
```

## Start the server

```zsh
$ yarn start
```

## Start sever in watch mode

```zsh
$ yarn watch
```
